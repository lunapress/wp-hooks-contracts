# WP Hooks Wrapper Contracts

Contracts and wrappers for WordPress Hooks API (actions & filters).

## Supported Functions

- `add_action()`
- `add_filter()`
- `remove_action()`
- `remove_filter()`
- `apply_filters()`
- `do_action()`
- `has_action()`
- `has_filter()`
- `current_action()`
- `doing_action()`
- `doing_filter()`
